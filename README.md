# Недельный отчет

Локальный прототип админки для недельного отчета. Сейчас факты, месяцы и события можно вести прямо в панели; Google Sheets остается опциональным мостом для будущей синхронизации.

## Что уже есть

- общий дашборд с одним горизонтальным графиком по месяцам и неделям;
- разрез `Все / МСК / СПБ` для KPI, графиков и месячной таблицы;
- режим `Обзор` с недельными столбиками и маркерами событий;
- режим `День` для быстрого ввода по дате, городу и метрике;
- режим `Месяц по неделям`, где можно добавить новый месяц и внести факт;
- отдельная вкладка `Сообщения` под будущую логику;
- режим `Карта событий` с календарем внешних факторов;
- файл `apps-script/AppsScript.gs` для Google Apps Script моста.

## Локальный запуск

```powershell
cd weekly-report-app
pnpm install
pnpm run dev
```

## Подключение Google Sheets

1. Открыть Google Sheets.
2. Открыть Apps Script.
3. Добавить код из `apps-script/AppsScript.gs`.
4. Запустить `setupWeeklyReportService`.
5. Запустить `setWeeklyReportPassword("ваш-пароль")`.
6. Опубликовать как Web App.
7. Добавить URL в `.env`:

```text
VITE_APPS_SCRIPT_URL=https://script.google.com/macros/s/.../exec
VITE_ADMIN_PASSWORD=ваш-пароль
```

Пока URL не задан, приложение работает на мок-данных локально.
